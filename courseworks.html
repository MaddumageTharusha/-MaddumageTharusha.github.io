GameInt - Interactive Guessing Game Mastermind

Python

Description:

Created 'GameInt' in Python, a guessing game where players try to guess a randomly generated 4-digit number between 1 and 6. With 8 chances, players receive instant feedback on each attempt. The result includes '1' for correct number and position, '0' for correct number in the wrong position, and '.' for incorrect numbers. The game awards points based on performance, allowing for replay with a new challenge.
